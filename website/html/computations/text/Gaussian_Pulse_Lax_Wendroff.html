Lax-Wendroff method for Gaussian Pulse demonstrates all the properties expected from its design: second-order convergence and monotonicity. That is all because all of the assumptions of the truncation error analysis are met in this case: the initial data is infinitely differentiable and gradients are not very steep.

Thus, Lax-Wendroff is suitable for sufficiently smooth initial data, although it fails for initial conditions that are not sufficiently smooth, which limits its practical utility, because it is usually necessary to model shock wave propagation accurately, which Lax-Wendroff cannot do.
